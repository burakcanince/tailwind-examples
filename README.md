## Installation

##### 1. Clone the repository
```bash
git clone https://github.com/burakcanince/tailwind-examples.git
```
##### 2. Go to the installation
```bash 
cd tailwind-examples
```

##### 3. Install the dependencies
```bash
npm install
```

##### 4. Start project
```bash
npm run serve
```
